MS Predictor V12 PWA
====================

Android:
1. Den Ordner auf GitHub Pages, Cloudflare Pages oder einen Webspace hochladen.
2. index.html auf dem Android-Handy in Chrome öffnen.
3. Chrome-Menü ⋮ -> "App installieren" oder "Zum Startbildschirm hinzufügen".

Wichtig:
- Die PWA läuft offline mit Demo-/lokalen Daten.
- Für echten API-Import auf Android ist ein kleiner Proxy nötig, weil API-Keys nicht direkt in einer öffentlichen PWA liegen sollten.
- Eine Cloudflare-Worker-Vorlage liegt als cloudflare-worker.js bei.
